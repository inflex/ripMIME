### Installation

1. Clone the project

    $ git clone https://github.com/inflex/ripMIME.git

2. Build it!

    $ make

For armv5: `make CC=arm-linux-gnueabi-gcc CFLAGS="-Wall --static -O2 -marm -march=armv5tej"`
